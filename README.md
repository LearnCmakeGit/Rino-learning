# Rino-learning

Step-by-step practice repo for learning Python, NumPy, PyTorch, Colab, GitHub workflow, and eventually a small RINO-style neural operator skeleton.

## Workflow goal

```text
GitHub source code
    ↓
Colab runtime
    ↓
run / train / infer
    ↓
commit changes back to GitHub
```

## Current exercises

- `day01/main.py`: Python basics and the write-run-modify-run loop.

## How to run Day01 in Colab

```python
!git clone https://github.com/LearnCmakeGit/Rino-learning.git
%cd Rino-learning
!python day01/main.py
```

## Two-week path

1. Python basics
2. NumPy matrix operations
3. PyTorch tensors
4. Autograd
5. Manual gradient descent
6. `nn.Module`
7. Dataset and DataLoader
8. Train / infer scripts
9. Encoder / decoder
10. RINO-mini skeleton
